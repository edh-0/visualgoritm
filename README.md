# 🎯 Algorithm Visualizer

Интерактивная визуализация алгоритмов сортировки на React

## 🚀 Демо
[Открыть демо](https://edh-0.github.io/visualgoritm)

## 📊 Доступные алгоритмы
- 🫧 Пузырьковая сортировка
- 🎯 Сортировка выбором  
- 📥 Сортировка вставками

## 🎮 Функционал
- Пошаговая анимация
- Управление скоростью
- Выбор алгоритма
- Подсветка элементов
- Генерация массивов
- Статистика шагов
